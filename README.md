# 맛.zip (Matzip AI) - 프로토타입

2025년 새싹 해커톤(SeSAC Hackathon) AI 서비스 프로토타입
![service ui]([https://drive.google.com/file/d/1dd6EfBupQ8LBZ_8ircLEQCIFQGjlJqLB/view?usp=sharing](https://drive.google.com/file/d/1dd6EfBupQ8LBZ_8ircLEQCIFQGjlJqLB/preview))

## 프로젝트 소개

**맛.zip**은 대화형 AI를 통해 사용자의 자연어 질문을 이해하고, 개인 취향과 상황에 맞는 맛집을 지도에서 추천하는 React Native 기반 모바일 서비스입니다.

## 주요 기능

### 1. 자연어 대화형 맛집 추천 🤖
- "애견 동반 가능한 브런치집 알려줘"
- "비오는 날 따뜻한 국물 요리 맛집 추천해줘"
- "첫 데이트하기 좋은 분위기 좋은 곳"

### 2. 실시간 지도 마커 표시 🗺️
- AI가 추천한 맛집을 **즉시 지도에 마커로 표시**
- 사용자 위치 기반 **거리순 자동 정렬**
- 현재 위치에서 각 맛집까지의 거리 표시 (km/m)

### 3. 실시간 내비게이션 🧭
- **내 위치에서 맛집까지 실시간 길찾기**
- 방향 화살표로 직관적인 안내 (⬆️ ↗️ ➡️ 등)
- 목적지까지의 거리와 방향 실시간 표시
- 채팅에서 "길찾기" 버튼으로 바로 내비게이션 시작

### 4. AI 맛집 정보 요약
- 수십 개의 리뷰를 3-4줄로 핵심만 요약
- 추천 메뉴, 매장 특징, 방문 팁 제공

### 5. 실시간 질의응답
- "주차 가능해?", "예약 필요해?", "아기의자 있어?"
- 즉석에서 궁금한 정보 확인

## 기술 스택

- **프레임워크**: React Native (Expo)
- **언어**: TypeScript
- **내비게이션**: Expo Router
- **지도**: WebView (카카오맵 지원 가능)
- **위치 서비스**: Expo Location
- **상태관리**: React Hooks

## 프로젝트 구조

```
sesac-prototype/
├── app/                        # 화면 및 라우팅
│   ├── (tabs)/                # 탭 네비게이션
│   │   └── index.tsx          # 메인 화면 (지도 + AI 채팅)
│   ├── restaurant/            # 맛집 상세 화면
│   │   └── [id].tsx          # 동적 라우팅
│   └── _layout.tsx            # 루트 레이아웃
├── components/                # 재사용 컴포넌트
│   ├── ChatInterface.tsx      # AI 채팅 인터페이스
│   └── KakaoMap.tsx          # 지도 컴포넌트
├── data/                      # Mock 데이터
│   └── mockRestaurants.ts    # 맛집 데이터 및 검색 로직
├── types/                     # TypeScript 타입 정의
│   └── restaurant.ts         # 맛집 관련 타입
└── package.json              # 프로젝트 의존성
```

## 설치 및 실행

### 1. 의존성 설치

```bash
pnpm install
```

### 2. 개발 서버 실행

```bash
# iOS 시뮬레이터
pnpm ios

# Android 에뮬레이터
pnpm android

# 웹 브라우저
pnpm web

# Expo Go로 실행
pnpm start
```

## 주요 화면

### 메인 화면
- 상단: 맛.zip 헤더 (추천된 맛집 수 표시)
- 중앙: 인터랙티브 지도
  - AI 추천 맛집 마커 실시간 표시
  - 사용자 위치 표시
  - 거리순 정렬된 맛집 리스트
  - 지도에서 직접 맛집 선택 가능
  - 내비게이션 패널 (활성화 시)
- 하단: AI 채팅 토글 버튼
- 팝업: AI 채팅 인터페이스 (60% 화면)
  - 각 맛집 카드에 "상세보기", "길찾기" 버튼

### 맛집 상세 화면
- 이미지 슬라이더
- 기본 정보 (이름, 평점, 주소, 영업시간, 가격대)
- 매장 특징 태그
- AI 요약
- 빠른 질문 버튼 (주차, 예약, 아기의자 등)
- 하단 액션 버튼 (길찾기, 전화, AI에게 묻기)

## Mock 데이터

프로토타입에는 5개의 샘플 맛집 데이터가 포함되어 있습니다:

1. **카페 도그런** - 애견동반 브런치 카페
2. **파스타 아모레** - 이탈리안 레스토랑
3. **한옥골 설렁탕** - 24시간 한식당
4. **스시 오마카세** - 고급 일식당
5. **베이커리 앤 카페** - 베이커리 카페

### AI 검색 키워드

채팅에서 다음 키워드를 사용하면 맞춤 결과를 받을 수 있습니다:

- "애견", "반려", "강아지" → 애견동반 가능한 곳
- "브런치" → 브런치 전문점
- "비", "국물", "따뜻" → 한식당
- "데이트", "분위기" → 평점 높은 곳
- "24시간", "야식" → 24시간 운영
- "파스타", "이탈리안" → 이탈리안 레스토랑
- "초밥", "스시", "일식" → 일식당
